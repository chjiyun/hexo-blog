D3.js绘制可交互的树状图

发表于 2018-03-17 更新于 2023-04-16 分类于 javascript ， 可视化

D3 是 Data-Driven Documents 的简称，意思是数据驱动的文档，他本质上是一个 JavaScript 函数库。D3.js 的原理 是使用HTML, SVG, CSS 来表现数据，用户将数据绑定到DOM，然后根据数据来操作文档，创建可交互的图形。

D3的优缺点

优点：

  1. 灵活，代码简洁，功能丰富
  2. 支持数据绑定到DOM，能直接操作DOM
  3. 布局layout丰富
  4. 配色好看，能激发创造的热情
  5. 基于SVG，缩放不会损失精度
  6. 快，基于数据的高校文档操作，性能良好
  7. 社区活跃，庞大的用户群

缺点：

  1. 上手较难，需要花一定的功夫
  2. 在大量数据面前性能不如Canvas，因为DOM操作会触发浏览器重排和重绘
  3. 兼容性较差，IE9以下不支持SVG
